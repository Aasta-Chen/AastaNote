Hexo server 顯示卡關

在安裝 Hexo 過程中，按照老師步驟，成功安裝 Node.js 和 npm (版本資訊有成功出現)，但輸入安裝 hexo server 指令時，卻出現 https://hexo.io/docs/ 的訊息(如圖)，無法訪問網站。 在 Discord 上發問，經過同學的熱心解答，我終於知道為什麼了

因為我少了 node_modules 資料夾，於是需要進行安裝，所以輸入 sudo npm install 後，就能順利安裝 node_modules，一安裝完成後，順利排除問題，可在本機上看到網站
