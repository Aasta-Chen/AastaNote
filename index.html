軟體工程師體驗營學習心得

當初為什麼報名這堂課？

剛開始學程式課程時，是在 Udemy 上看到的，於是入坑，從那之後，只要是想學程式，第一個就是想到六角 前幾年我處於人生最低潮的時候，工作上不順，又遇到疫情，那段時間無法專心學習程式，最後在各方面的壓力之下，就爆掉了 至此，再也不碰程式。

後來從平面設計轉職為網頁設計師，工作上還是都會碰到點程式，就是很基本的 CSS，可是很生疏加上排斥不想跟程式親近 直到今年忘了什麼管道，看到了六角的體驗營的課程，那時其實還不確定自己心裡是否已經不排斥程式了。

可覺得自己空了滿久了，加上工作我也希望可以有突破，體驗營，就當自己先初步的去練程式，慢慢的把過去學習的抓回來 這次發現自己好多了，然後要求自己把每日任務、Flex 十個關卡、直播班任務以及課前預習、課後練習全部要做完，盡量做到確實 包含這次校長直播課中，現場切版，使用 Emmet，在兩分多鐘切完，我又多了一項要練習的是快捷鍵要記要練。

第一周學習到的是
重新抓回過去的記憶，比如 VS Code 及套件、課前預習、課後練習 另外訂下的目標：Emmet 要到熟練，Code 要乾淨 這次上課很有感的是因為現在公司老闆是全端出身，所以無形中看校長上課，莫名的好像看到了老闆 平常上班，老闆會要求這些部分，也很重視

重新練習過程中，才發現無形中也被影響，比如 Code 上會想盡量整理乾淨，因為會影響到網站讀取速度，維護上也比較好維護，包含圖片的檔案大小….等 因為這樣，所以平常對於客戶，我也會想這樣要求客戶要盡量每個動作做到確實。

第二周學習到的是
Flexbox，這也是以前我學習過的，但是以前不會一直想去練習，這次因為自己要好好認真學習，於是課前預習完後，就是十關的 Flexbox 認真的做， 每一關都畫圖，及思考應該如何命名、以及決定主軸、交錯軸是以資訊流決定等等，剛開始會卡關，發現自己觀念上不夠清楚，就會回頭重新看影片，思考為什麼， 到四、五關過後，我做的速度就快滿多。

做完 Flexbox 精神時光屋後，回頭接著做每日任務，其中之一任務 『 jQuery 基礎教學＋點選按鈕動態切換 active 效果 』，這又是另一塊知識點，這個學完也許我猜，才有辦法做出直播班任務二『 LV 3｜做所有 RWD 頁面，且 JS 動畫效果都做 』

第三周學習到的是
標題寫著『 RWD 一點都不難，手把手打造手機版網頁 』，我又卡關了，做著每日任務才發現原來自己觀念並沒有到很清楚，心裡默默想，RWD 很難對現階段的我來說，沒有一點都不難

所以光是這兩周至第三周中間，我就學了滿滿好多的知識點

第四周學習到的是
Hexo 極速免費架站全攻略，如何操作終端機打指令，GitHub Pages，我終於懂了怎麼把內容推到 GitHub 上了 記得前幾年的直播課，不知道是我不認真還是沒有特別相關的影片教 GitHub 這一塊，所以即使組員教我，我還是無法學好，也沒有成功推上 這次這個知識點對我來說是非常意外的收獲

到目前一路上，多次卡關，比如在每日任務的 RWD、透過終端機安裝 Hexo 的相關環節，將內容推上 GitHub 又遇上問題等，兩個小時左右的直播影片，硬是看了四個多小時，因為一路上遇到問題，於是重新再操作，思考為什麼不行等等，當然在過程中，在六角頻道 Discord 上發問，有熱心同學幫忙解答，真的是超級感謝

但默默的突然也發現我也算某程度完成了其中一關的任務，只是對於頁面上，我想再編輯的更完整一些，再做提交 過程中，不時的卡關，接著一關一關的突破，都讓我很開心

如今也即將邁入第五周，而後面我還有好多事要做，比如客製化我的網頁，另外釋出的 jQuery、BS5、JS，這些都是當初我在爆掉的那年應該要學好的，現在我會一個一個再學起來，再學 SCSS 相關的知識

最後，以上滿滿五周，我收獲滿滿，有好多的知識點學習，與我當初還沒報名前想，只是先試水溫，是不是自己在程式這塊好多了 因為想像中的體驗營就是體驗，沒想到超乎我想像，有滿滿的知識點，我覺得對我來說不只是體驗營，是物超所值紮實體驗營。

最喜歡直播班的哪些活動？

最喜歡聽校長直播，樂於分享及真性情表現，印象最深是卡斯伯老師部落格的 console.log。直播雖然一周一場，但說真的覺得時間不夠用，天天下班就想著趕快回家寫作業，想辦法破解卡關的地方

如果時光能倒流，會希望自己再次注意哪些細節？

沒有特別想到，因為每天都過的紮實，唯一覺得不夠的是時間，看到眼睛都覺得累到快睜不開，因為為了解卡關的地方，就會想有沒有什麼樣的資料可以破解及盡量釐清自己不懂的環節。

覺得還可以改善的細節

Emmet 練習、以校長為目標，也可以打出清晰乾淨的架構、class 命名、Code 可以加強整理，乾乾淨淨的，切版可以切到與設計稿一模一樣

身為學長姐，分享給想入坑的新同學一些勉勵的話

六角是個非常好的學習環節，校長、副校長都是樂於分享的老師，助教非常認真的改 Code，提供滿滿的建議，如果真的想學習的人，真的很推！！！

結語

就是每天下班想著回家練習寫作業，即使課程結束，後續也會不斷的練習切版相關 目前體驗營課程結束後，要繼續迎戰校長推坑的 AAPD 課程，一路就到八月，接著會接回六角學院的課程(如果有合適課程的話)，繼續紮實的練功。

非常感謝老師以及助教們，超級用心與辛苦！
